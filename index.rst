=====================================
Welcome to Zope Project and Community
=====================================

Zope is a free and open source web application server written in the
object-oriented programming language "Python".  Since it's release in
1998 Zope continued to grow into many distinct applications,
frameworks, libraries and tools.

:doc:`Learn more <world>`

Zope Community
==============

The Zope community is one of the largest and most professional
open-source communities worldwide.

:doc:`Learn more <community>`

Zope Foundation
===============

The Zope Foundation has the goal to promote, maintain, and develop the
Zope platform.

:doc:`Learn more <foundation>`

Github
======

Zope community projects are hosted in the `Zope Foundation
organization <https://github.com/zopefoundation>`_ on GitHub.
